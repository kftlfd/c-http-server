Minimal HTTP Server in C

A small experimental HTTP server written in pure C, built to explore how HTTP servers actually work + get some practice with C.

  • Non-blocking I/O with poll()
  • Per-client state machine
  • HTTP/1.1 keep-alive
  • Basic request parsing
  • Static file serving (GET / HEAD)

It’s intentionally minimal, has lots of limitations, and not production-ready.
